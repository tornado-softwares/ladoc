# Usage

I don't even like writing documentation just take this

| Markdown | Result |
| --- | --- |
| `I **hate** writing documentation` | I **hate** writing documentation |
| `I *hate* writing documentation` | I _hate_ writing documentation |
| `I _hate_ writing documentation` | I _hate_ writing documentation |
| `I **_hate_** writing documentation` | I **_hate_** writing documentation |
| `I ~hate~ love writing documentation` | I ~_hate_~ love writing documentation |
| `I hate writing [documentation](https://en.wikipedia.org/wiki/Documentation)` | I hate writing [documentation](https://en.wikipedia.org/wiki/Documentation) |
