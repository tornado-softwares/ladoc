# Usage

To quote your favorite philosopher

```md title="Markdown Example"
> "dubito, ergo cogito, ergo sum.."
```

> "dubito, ergo cogito, ergo sum.."

## Nesting

You can also nest quotes and use markdown syntax in it.

````md title="Markdown Example"
> "dubito, ergo cogito, ergo sum.."
>
> > ```mermaid
> > flowchart LR
> >    A[René Descartes]
> >
> >    A --> B[Personal Information]
> >    B --> B1[Born: March 31, 1596]
> >    B --> B2[Born in France]
> >    B --> B3[Died: February 11, 1650]
> >    B --> B4[Died in Stockholm, Sweden]
> >
> >    A --> C[Occupations]
> >    C --> C1[Philosopher]
> >    C --> C2[Mathematician]
> >    C --> C3[Scientist]
> >
> >    A --> D[Famous Ideas]
> >    D --> D1["I think, therefore I am"]
> >    D --> D2[Rationalism]
> >    D --> D3[Mind-Body Dualism]
> >
> >    A --> E[Major Works]
> >    E --> E1[Discourse on the Method]
> >    E --> E2[Meditations on First Philosophy]
> >    E --> E3[Principles of Philosophy]
> >
> >    A --> F[Contributions]
> >    F --> F1[Founder of Modern Philosophy]
> >    F --> F2[Analytic Geometry]
> >    F --> F3[Cartesian Coordinate System]
> >
> >    A --> G[Legacy]
> >    G --> G1[Influenced Philosophy]
> >    G --> G2[Advanced Mathematics]
> >    G --> G3[Shaped Scientific Thinking]
> > ```
````
